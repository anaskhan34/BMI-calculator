# BMI Calculator

A simple **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
It takes **weight (in kg)** and **height (in cm)** as input and calculates your BMI with a category (Underweight, Normal, Overweight, Obese).
